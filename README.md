# N.E.R.D.S. Landing Page

Landing page institucional do grupo N.E.R.D.S. (UFC), construída com React, Vite, TypeScript, React-Bootstrap e SCSS, com estética cyber-tech dark.

## Documentação Oficial (Fonte da Verdade)

Leia estes arquivos antes de alterar código:

- `docs/PRODUCT.md` — regras de negócio, escopo e requisitos
- `docs/ARCHITECTURE.md` — arquitetura, estrutura e padrões técnicos
- `docs/DESIGN_SYSTEM.md` — diretrizes visuais e comportamento de UI
- `.cursorrules` — regras operacionais obrigatórias para IA

## Comandos principais

- `npm install`
- `npm run dev`
- `npm run lint`
- `npm run build`
